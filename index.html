Vanilla Accordion — Lightweight, dependency-free, ADA-friendly

A tiny, dependency-free Vanilla JavaScript accordion that’s accessible by default. Automatically sets ARIA attributes, adds keyboard navigation, emits open/close events, and can generate JSON-LD schema for SEO. Easy to style and customize.

Theme None

Setting theme to empty disables access to all styles options. This is a barebones accordion.

{
  theme: ""
}

Theme Core

Core delivers the essential accordion layout with inline styles only—no external CSS needed. Ideal for rapid prototypes or quick drop-ins, it works out of the box and can be overridden if custom styling is required.

By default, it doesn't include icons for the open/close states. You can add your own—using inline SVG, an <img> element, or even emojis. 

{
  theme: "core",
  icons: ['<svg>...</svg>', '<svg>...</svg>'],
  iconClass: 'jsa-icon',
  termPadding: '0.5em 1em 0.5em 0'
}

Theme Minimal

Minimal provides a clean, lightweight starting point for custom styling. It applies only essential base styles and is meant to be extended or overridden for your project's unique look.

Requires: jsa-themes.min.css or add _jsa.theme-minimal.scss to your project.

{
  theme: "minimal",
}
